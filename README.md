VO2max calculation based on George et al. (1993) algorithms.

V02max = 100.5 + 8.344 * GENDER (0 = female; I = male) - 0.1636 * BODY MASS (kg) - 1.438 * JOG TIME (min 1 mile) - 0.1928 * HEART RATE (bpm)
